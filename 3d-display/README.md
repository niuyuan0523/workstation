# 3D晶体结构显示系统

一个基于Three.js的交互式3D晶体结构可视化系统，用于展示不同晶系的立方体及其对称面。

## 功能特性

### 🔮 支持的晶体类型
1. **菱形十二面体立方体** - 立方晶系，9个对称面
2. **六方双锥** - 六方晶系，7个对称面  
3. **四方双锥** - 四方晶系，5个对称面
4. **双斜方柱** - 斜方晶系，3个对称面

### ✨ 显示效果
- **高质量3D渲染**：使用PBR材质和阴影效果
- **精细的边缘线条**：突出晶体结构
- **装饰性细节线条**：增强视觉效果
- **现代化UI界面**：毛玻璃效果和渐变按钮

### 🎛️ 交互功能
- **坐标轴显示**：3D坐标轴与箭头指示
- **对称面可视化**：根据晶系显示相应数量的对称面
- **鼠标控制**：旋转、缩放、平移视角
- **实时信息面板**：显示当前晶体的详细信息

## 技术栈

- **Three.js** - 3D图形渲染
- **Vite** - 构建工具
- **ES6 Modules** - 模块化开发
- **CSS3** - 现代化样式

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 对称面说明

### 立方晶系（9个对称面）
- 3个主要对称面：(100), (010), (001)
- 6个对角对称面：(110), (101), (011)及其对称面

### 六方晶系（7个对称面）
- 1个水平对称面
- 6个垂直对称面，每60°一个

### 四方晶系（5个对称面）
- 1个水平对称面
- 4个垂直对称面

### 斜方晶系（3个对称面）
- 3个互相垂直的对称面

## 使用说明

1. **选择晶体**：点击顶部按钮切换不同的晶体类型
2. **显示坐标轴**：点击"显示坐标轴"按钮
3. **显示对称面**：点击"显示对称面"按钮查看对称面
4. **视角控制**：
   - 鼠标左键拖拽：旋转视角
   - 鼠标滚轮：缩放
   - 鼠标右键拖拽：平移视角

## 开发说明

每个晶体模型都包含：
- 精细的几何体定义
- 高质量的材质和光照
- 边缘线条突出结构
- 装饰性细节线条
- 阴影效果

对称面根据晶系类型自动生成，使用半透明材质和不同颜色区分。

## 浏览器兼容性

支持所有现代浏览器，需要WebGL支持。
